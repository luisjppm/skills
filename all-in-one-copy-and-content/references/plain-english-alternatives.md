# Plain-English Alternatives

## Contents

- Scope
- High-Impact Replacements
- Extended A-Z List
- Phrases to Delete Entirely

## Scope

Use for:

- simplifying copy after structure and message are already correct
- replacing legalistic, corporate, or inflated wording with plain language

Do not use for:

- removing necessary technical precision in domain-specific copy
- rewriting positioning or offer strategy

Usage order:

1. Run content logic and clarity edits first.
2. Replace complex terms with plain alternatives.
3. Remove filler phrases that add no meaning.
4. Re-read for tone to avoid oversimplifying nuanced claims.
