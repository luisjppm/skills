# all-in-one-copy-and-content

Comprehensive copy and content skill for modern marketing work: strategy,
ideation, drafting, conversion copywriting, and systematic editing in one
workflow.

## What this skill does

- Plans content strategy with pillars, clusters, and prioritization.
- Drafts blog, social, and newsletter content using channel-specific formats.
- Writes conversion-focused web copy for core page types.
- Edits existing copy with structured quality sweeps.
- Improves readability with transition guidance and plain-language rewrites.

## When to use it

Use this skill when you need help with:

- creating homepage, landing, pricing, feature, or about page copy
- improving existing copy for clarity, proof, and conversion
- deciding what content to publish and in what order
- writing blog posts, threads, LinkedIn posts, or newsletters
- polishing tone, flow, and plain-language quality

## Why it is useful

- One system covers strategy, production, and editing end-to-end.
- Reference files are split by scope for faster loading and cleaner routing.
- Includes clear mode contracts to reduce overlap and improve consistency.

## Quick prompt examples

- "Build a 90-day content strategy for our B2B SaaS."
- "Write a LinkedIn post and email newsletter from this product update."
- "Create homepage copy with three headline and CTA options."
- "Edit this landing page copy using a structured sweep process."
- "Simplify this copy and improve transitions without changing meaning."

## File map

- `SKILL.md`: Core orchestration workflow, routing, and output contracts.
- `references/content-strategy-playbook.md`: Strategy framework, ideation, and prioritization model.
- `references/content-creation-formats.md`: Drafting templates and engagement patterns for editorial channels.
- `references/conversion-copywriting-playbook.md`: Conversion page structures, section guidance, and CTA frameworks.
- `references/copy-editing-seven-sweeps.md`: Structured editing workflow with quick-pass checks.
- `references/natural-transitions.md`: Transition phrase library and anti-pattern guidance.
- `references/plain-english-alternatives.md`: Plain-language substitutions and filler-phrase removals.

## Important implementation note

Use one primary reference file per mode. Add secondary files only when needed,
especially during polish passes, to keep context lean and avoid mixing scopes.
