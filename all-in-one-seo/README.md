# all-in-one-seo

Comprehensive SEO skill for modern search: traditional SEO, AI/AEO visibility,
programmatic SEO, technical implementation, structured data, and content
quality controls.

## What this skill does

- Runs full technical and on-page SEO audits.
- Plans and validates programmatic SEO rollouts at scale.
- Improves AI citation visibility (AI Overviews, ChatGPT, Perplexity, Copilot, Claude).
- Applies structured data patterns and validation workflows.
- Enforces E-E-A-T and editorial quality guardrails.

## When to use it

Use this skill when you need help with:

- SEO audits and ranking drops
- robots.txt, sitemaps, canonicals, metadata, Core Web Vitals
- schema markup and rich-result readiness
- AI SEO / AEO / GEO / LLMO strategy and monitoring
- scaling SEO pages with templates and data

## Why it is useful

- One workflow covers both traditional search and AI answer engines.
- Reference files are split by scope for fast loading and low context bloat.
- Includes both strategy playbooks and implementation-level examples.

## Quick prompt examples

- "Run a full SEO audit for my site and prioritize fixes by impact."
- "Build a 90-day SEO + AEO roadmap for a B2B SaaS."
- "Design a safe programmatic SEO rollout for [keyword pattern]."
- "Improve this page to increase AI citation likelihood."

## Important implementation note

For schema validation, do not rely on static fetch output alone. Use rendered
checks such as Google Rich Results Test or runtime inspection because many
sites inject JSON-LD with JavaScript.
